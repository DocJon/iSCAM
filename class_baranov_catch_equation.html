Baranov catch equation solution for 1 or more fleets. 

The following function solves the Baranov catch equation for multiple fleets using
a Newton-Raphson alogrithm to find a vector of fishing mortlity rates (ft) that 
predicts the total catch for each fleet.  The Jacobian matrix is computed and 
subsequently inverted to compute the Newton step for each fishing rate.

BARANOV CATCH EQUATION:

Parameters
cta vector of observed catches, 1 element for each gear.
minstananeous natural mortality rate.
Va matrix of selectivities (row for each gear, col for each age)
naa vector of numbers-at-age at the start of each year
Returns
Returns a vector of instantaneous fishing mortality rates.
